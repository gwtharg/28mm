Markdown Notes

Tables

You can create tables by assembling a list of words and dividing them with hyphens - (for the first row), and then separating each column with a pipe |:

so this


First Header | Second Header
------------ | -------------
Content from cell 1 | Content from cell 2
Content in the first column | Content in the second column

Renders as this

First Header Second Header
Content from cell 1 Content from cell 2
Content in the first column Content in the second column

Unordered Lists

To create an unordered list, add dashes (-), asterisks (*), or plus signs (+) in front of line items. Indent one or more items to create a nested list.

Links with names

To create a link, enclose the link text in brackets (e.g., [Duck Duck Go]) and then follow it immediately with the URL in parentheses (e.g., (https://duckduckgo.com)).

A better search engine would be [Duck Duck Go](https://duckduckgo.com)

A better search engine would be Duck Duck Go

Raw Links and Email Addresses

To create a link or an email link , enclose the link text in angled brackets

https://duckduckgo.com or yourname@home.com

<https://duckduckgo.com> or <yourname@home.com>
