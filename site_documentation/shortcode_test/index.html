rawhtml shortcode

This allows us to enter html within a shortcode and run within a markdown file. We now can leverage html to enhance our markdown. To show the code here we had to used an Entity Encoder. Entity Encoder

This is the code and the rawhtml shortcode to allow html colours in our example.

{{< rawhtml>}}

       <h2>HTML embedded in Markdown</h2>  
      Some  text with <span style="color:blue"> Simple <b>blue</b> coloured text.<br>
      </span>Back to normal text. Now we can highlight in <span style="color:red"> <b>red<> /b> this is red.</span><br>

{{< /rawhtml >}}
