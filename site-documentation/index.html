Hugo TOML YAML Mix

config.toml uses TOML format:

title = “My New Hugo Site”

but archetypes/default.md uses YAML

title: “Test index”

https://github.com/gohugoio/hugo/issues/5241

The reason is GitHub’s support for YAML front matter. Currently they do not support TOML front matter (Metadata).

So we are going to use **TOML** for the **config.toml** file then **YAML** for the rest of the contents.
